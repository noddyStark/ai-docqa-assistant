# AI-Powered Documentation Q&A Assistant

A backend AI knowledge assistant built with Java and Spring Boot using Retrieval-Augmented Generation (RAG) concepts to answer developer questions over technical documentation.

## Tech Stack
- Java 21
- Spring Boot
- Maven
- REST APIs
- IntelliJ IDEA

## Current Progress
- Spring Boot project initialized
- Health check endpoint added
- Base package structure created

## Upcoming Features
- Document ingestion API
- Chunking pipeline
- Embeddings integration
- Vector search
- LLM-based Q&A with source attribution
